# Запуск на iPhone — 3 клика

## Xcode уже открыт

### Клик 1: Team
```
Signing & Capabilities → Team → Твой Apple ID
```

### Клик 2: iPhone
```
Верхняя панель → Выбери свой iPhone (не симулятор)
```

### Клик 3: Run
```
▶ (Cmd+R)
```

---

## Что увидишь

```
┌─────────────────────────────────────────────┐
│  [ Контакты ] [ Папки ] [ Юнона ] [ Настройки ]  │
└─────────────────────────────────────────────┘
```

**Юнона** — центральная вкладка (открывается по умолчанию)

|---------|------------|--------------|

## Если "Untrusted Developer"

На iPhone:
```
Настройки → Основные → VPN и управление устройством → Доверять
```
