## Как скачать Montana без App Store

Это инструкция. Она работает. Мы делаем это для своего приложения. ТБанк может сделать то же самое для всех своих.

### Шаг 1. Откройте Safari на iPhone

Только Safari. Другие браузеры не работают — это ограничение Apple.

Перейдите на:

```
montana.app/ios
```

### Шаг 2. Скачайте профиль

Нажмите кнопку «Установить Montana».

Safari предложит скачать профиль. Это файл `.mobileconfig` — стандартный формат Apple для конфигурации устройств.

Появится надпись: «Профиль загружен».

### Шаг 3. Установите профиль

Откройте **Настройки** → **Основные** → **VPN и управление устройством**.

Вы увидите «Профиль Montana». Нажмите на него.

Нажмите **Установить**. Введите код-пароль устройства.

Профиль отправит на наш сервер уникальный идентификатор вашего устройства — UDID. Это нужно чтобы подписать приложение именно для вашего iPhone.

### Шаг 4. Подождите

Сервер регистрирует ваше устройство. Генерирует сертификат. Подписывает приложение.

Обычно это занимает до двадцати четырёх часов. Иногда быстрее.

Вы получите уведомление когда всё будет готово.

### Шаг 5. Скачайте приложение

Вернитесь на montana.app/ios.

Нажмите «Скачать Montana».

iOS установит приложение. Готово.

## Почему это работает

Apple не может закрыть этот механизм. Это их собственная инфраструктура:

- **Enterprise Distribution** — для корпоративных приложений
- **Ad-Hoc Distribution** — для тестирования и ограниченного распространения
- **Developer Certificates** — для разработчиков

Компании по всему миру используют это для внутренних приложений. Apple использует это для бета-тестирования. Закрыть этот канал — значит сломать собственную экосистему.

## Если сертификат отзовут

Это может случиться. Apple периодически отзывает сертификаты.

Приложение перестанет открываться. Вы увидите сообщение что разработчик не доверен.

Решение простое:
1. Удалите старый профиль из настроек
2. Повторите процедуру с начала
3. Получите новый сертификат
4. Скачайте приложение заново

Неудобно. Но работает. И никто не может это остановить.

## Что это значит для ТБанк

ТБанк может сделать то же самое:

1. Создать страницу tbank.ru/ios
2. Генерировать профили для каждого клиента
3. Выпускать сертификаты с согласия клиента
4. Подписывать все приложения холдинга — ТБанк, Т-Мобайл, Т-Страхование, Т-Бизнес, Т-Инвестиции
5. Раздавать напрямую

Клиент один раз устанавливает профиль. Получает все приложения. Бесплатно. От своего банка. Без сторонних магазинов.

Каждое приложение — отдельный сертификат. Отзыв одного не влияет на другие.
