## Почему так?

### App Store (Junona VPN)
| За | Против |
|---|---|
| VPN легальны в App Store | Ревью Apple |
| NordVPN, ExpressVPN — примеры | Могут отклонить по другим причинам |
| Якорь для новых пользователей | |
| Доверие "официального" приложения | |

### Сертификаты (Wallet + Contracts)
| За | Против |
|---|---|
| Apple не контролирует | Нужна регистрация UDID |
| Нет ревью | Доверие сертификату |
| Быстрые обновления | Может быть отозван |
| Крипто-функции без ограничений | |

## Потоки пользователя

### Новый пользователь
```
1. Находит Junona VPN в App Store
2. Устанавливает
3. Подключает VPN (получает защиту + начисление Ɉ)
4. Видит "Установить Wallet" в приложении
5. Переходит на install.montana.network
6. Устанавливает Wallet + Contracts
7. Все приложения работают вместе
```

### Опытный пользователь
```
1. Заходит на install.montana.network
2. Устанавливает все 3 приложения сразу
3. Junona также доступна в App Store (backup)
```

## Взаимодействие приложений

```swift
// Из Junona (App Store)
MontanaLinks.openWallet()           // Открыть Wallet (если установлен)
MontanaLinks.openContracts()        // Открыть Contracts

// Проверка установки
if MontanaLinks.isInstalled(.wallet) {
    // Wallet установлен
} else {
    // Показать кнопку "Установить"
}

// Общий Keychain
MontanaKeychain.shared.walletAddress  // Доступен из всех приложений
```

## Резервные варианты

### Если Apple удаляет Junona VPN
```
1. Wallet и Contracts продолжают работать
2. VPN доступен через install.montana.network
3. Пользователи получают уведомление
```

### Если сертификаты отзывают
```
1. Junona VPN остаётся в App Store
2. Функции Wallet/Contracts интегрируются в Junona
3. Или новый Enterprise-аккаунт
```

### Если всё блокируют
```
1. PWA: app.montana.network
2. Android: Play Store + APK
3. Desktop: macOS/Windows apps
```

## App Store Guidelines Compliance

### Junona VPN соответствует:

**4.2.2 — VPN Apps**
- ✅ Используем NetworkExtension API
- ✅ Не обходим системные функции
- ✅ Чёткое описание функций

**5.4 — VPN Content**
- ✅ Не используем для нелегального контента
- ✅ Постквантовое шифрование — легально
- ✅ Нет логирования трафика

**3.1.1 — In-App Purchase**
- ✅ VPN бесплатный (монетизация через присутствие)
- ✅ Не требуем оплаты за базовые функции

### Что НЕ включаем в App Store версию:
- ❌ Криптовалютные переводы
- ❌ Кошелёк с балансом
- ❌ Контракты
- ❌ Любые финансовые функции

Эти функции — в Wallet/Contracts через сертификаты.
