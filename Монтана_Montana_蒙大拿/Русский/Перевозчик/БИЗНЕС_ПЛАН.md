# Сова — Морская Фрахтовая Платформа

## Бизнес-план

---

## Концепция

**Сова** — B2B платформа для морских перевозок:
- Судовладельцы
- Фрахтователи
- Брокеры
- Агенты

**Авторизация:** Telegram бот @junomontanaagibot

---

## Шаг 1: Идея и рынок

### Проблема
Морская логистика непрозрачна. Брокеры работают через звонки и email. Нет единой платформы.

### Решение
Цифровая платформа с:
- Картой судов в реальном времени
- Листингом грузов
- Прямым контактом
- Умным мэтчингом (AI)

### Конкуренты
| Платформа | Слабость |
|-----------|----------|
| MarineTraffic | Только трекинг |
| Xeneta | Только аналитика |
| Flexport | Только контейнеры |

**Преимущество Совы:** Полный цикл + AI + криптовалюта Ɉ

---

## Шаг 2: Продукт (MVP)

### Функционал v1.0

1. **Авторизация через Telegram**
   - Бот @junomontanaagibot
   - Верификация компании
   - Роли: судовладелец / фрахтователь / брокер / агент

2. **Карта судов**
   - AIS данные
   - Статус: свободно / в рейсе
   - Характеристики: DWT, тип, флаг

3. **Листинг грузов**
   - Маршрут: откуда → куда
   - Объём, ставка, срок
   - Срочность

4. **Сделки**
   - Предложение / контрпредложение
   - Чат между сторонами
   - Смарт-контракт (Montana Protocol)

5. **Профиль**
   - Рейтинг
   - История сделок
   - Документы

### Технологии
- **iOS:** SwiftUI
- **Backend:** Montana Protocol (72.56.102.240)
- **Крипто:** Ɉ для комиссий
- **AI:** Юнона для мэтчинга
