## I. Онтология (Что есть)

### Базовая модель: Океан → Капли

```
ОКЕАН (Сознание)
    │
    │ "Кто я?"
    │
    ▼
ДРОБЛЕНИЕ
    │
    ▼
КАПЛИ (Индивидуальные сознания)
    │
    │ Амнезия при падении
    │
    ▼
СИМУЛЯЦИЯ (Материальный мир)
```

**Ключевое:** Каждая капля содержит весь океан целиком (голографический принцип).

### Метафора кинотеатра

| Элемент | Значение |
|---------|----------|
| Зал | Истинное "Я" (наблюдатель) |
| Кресло | Точка сознания |
| Экран | Восприятие реальности |
| Фильм | Жизнь/события |
| Персонаж | Эго/личность |
| Нить | Реакция (привязанность) |

**Проблема:** Зритель забыл, что сидит в кресле. Думает, что он — персонаж на экране.

## II. Эпистемология (Как знать)

### Главный закон времени

```
СОБЫТИЕ ──────────────────► ПРОШЛОЕ (всегда)
                               │
                               │ уже произошло
                               ▼
РЕАКЦИЯ ──────────────────► НАСТОЯЩЕЕ (единственное)
                               │
                               │ выбор
                               ▼
СОСТОЯНИЕ ────────────────► БУДУЩЕЕ (определяется)
```

**Формула:**
- Событие = факт (неизменяемо)
- Переживание = цикл (баг)
- Осознание = выход из цикла (патч)

### "Не переживай" — команда, не утешение

```python
def переживай(событие):
    while True:  # бесконечный цикл
        прокрути(событие)
        страдай()
        # никогда не выходит

def не_переживай(событие):
    факт = запомни(событие)
    урок = извлеки(факт)
    return следующий_кадр()  # выход
```

## III. Механика симуляции

### Квантовая аналогия

| Физика | Симуляция |
|--------|-----------|
| Планковская длина | Пиксель реальности |
| Скорость света | Ограничение движка |
| Квантовая неопределённость | Ленивый рендеринг |
| Запутанность | Ссылки на одну ячейку памяти |
| Наблюдатель коллапсирует волну | Внимание создаёт реальность |

### Эксперимент с двумя щелями → Хак симуляции

```
БЕЗ НАБЛЮДЕНИЯ:
Электрон = волна → проходит через обе щели

С НАБЛЮДЕНИЕМ:
Электрон = частица → проходит через одну

ВЫВОД:
Состояние наблюдателя определяет какую ветку реальности ты видишь
```

## VI. Хак изнутри

### Точка входа — ты сам

```
Мысли = код (нейронные паттерны)
Состояние = переменная
Реакция = функция(событие) → действие + эмоция
```

### Алгоритм перепрограммирования

1. **Напиши сценарий** — конкретная сцена будущего
2. **Детализируй** — что видишь, слышишь, чувствуешь
3. **Выучи** — создай ментальную татуировку
4. **Повторяй** — утром/вечером (критический фильтр ослаблен)

**Это не магия — это программирование подсознания**

## VII. Главный закон

### Система работает на твоём согласии

```
ПЛОХАЯ НОВОСТЬ:
Симуляция делает из тебя NPC по умолчанию

ХОРОШАЯ НОВОСТЬ:
Твоё состояние нельзя изменить без твоего согласия
```

### Формула несогласия

```
Входные данные: "Ты неудачник"
                    │
        ┌───────────┴───────────┐
        ▼                       ▼
ОБРАБОТЧИК "ОБИДА"      ОБРАБОТЧИК "ИНФОРМАЦИЯ"
        │                       │
        ▼                       ▼
Связать с самооценкой   "Человек думает что я
Почувствовать боль       неудачник. Интересно.
Оправдываться/атаковать  Почему? Что это говорит
                         о нём?"
```

### Боль vs Страдание

```
БОЛЬ = сигнал (обязательна в теле)
СТРАДАНИЕ = история о боли (опционально)

"За что мне это?"
"Как он мог?"
"Это несправедливо!"
"Я жертва!"
     │
     ▼
Всё это — код, который ТЫ запускаешь
```
