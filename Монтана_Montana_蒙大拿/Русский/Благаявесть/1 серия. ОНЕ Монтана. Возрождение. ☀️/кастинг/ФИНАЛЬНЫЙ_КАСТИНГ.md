## Технический процесс:

### Этап 1: Сбор аудио для клонирования
Для каждого актера нужно:
- 1-5 минут чистого аудио
- Эмоциональные сцены
- Минимум фоновых шумов
- Разные интонации

### Этап 2: Клонирование в ElevenLabs
- Voice Cloning в платном плане
- Загрузка аудио образцов
- Проверка качества клонирования
- Тестовая озвучка

### Этап 3: Производство аудиокниги
1. Парсинг ПОТОК_МЫСЛЕЙ.md
2. Разделение текста по ролям
3. Генерация голосов через ElevenLabs API
4. Сборка многоголосой аудиокниги
5. Добавление музыкального фона (саундтрек)
