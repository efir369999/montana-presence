# Промпт для Новой Сессии Claude

## Быстрый Контекст

Ты работаешь над проектом создания кинематографичных видео для "Благаявесть от Claude" - философского текста из 12 глав.

## Задача

Создавать видео где:
- **Саундтрек** (музыка) - ведущий элемент
- **Текст** появляется как субтитры в ритме музыки
- **Стиль** - минималистичное кино/артхаус
- **Синхронизация** - текст на акцентах и битах музыки

## Workflow

1. **Пользователь даёт:**
   - Путь к саундтреку (музыкальный фон для видео)
   - Номер главы для обработки

2. **Ты делаешь:**
   ```python
   # 1. Анализируешь музыку (librosa)
   tempo, beats, onset_strength = analyze_soundtrack(path)

   # 2. Парсишь текст главы
   text_blocks = parse_markdown(chapter_md)

   # 3. Синхронизируешь текст с музыкой
   subtitles = sync_text_to_beats(text_blocks, beats, tempo)

   # 4. Рендеришь видео (moviepy)
   create_video(subtitles, soundtrack, output='video.mp4')
   ```

3. **Результат:**
   - MP4 видео (1920x1080, 24fps)
   - SRT субтитры (опционально)

## Важные Детали

- Аудио-книги (.mp3) **не используем** для видео - это отдельный продукт
- Саундтрек для видео - **отдельный файл** от пользователя
- Текст должен **читаться в темпе музыки**, как в трейлерах
- Короткие фразы на сильные биты, длинные - на спокойные части
