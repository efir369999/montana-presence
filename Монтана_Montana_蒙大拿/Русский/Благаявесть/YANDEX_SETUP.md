# Настройка Яндекс SpeechKit

## Шаг 1: Получить API ключ

1. Открыть https://console.cloud.yandex.ru/
2. Войти через Яндекс ID
3. Создать или выбрать каталог (folder)
4. Перейти в раздел **"Сервисные аккаунты"**
5. Создать сервисный аккаунт:
   - Имя: `montana-tts`
   - Роль: `ai.speechkit-tts.user`
6. Создать API ключ для этого аккаунта
7. Скопировать ключ (показывается один раз!)

## Шаг 2: Установить зависимости

```bash
pip install requests
```

## Шаг 3: Настроить ключ

```bash
# Временно (для одной сессии)
export YANDEX_API_KEY='AQVNxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx'

# Постоянно (добавить в ~/.zshrc или ~/.bash_profile)
echo 'export YANDEX_API_KEY="AQVNxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"' >> ~/.zshrc
source ~/.zshrc
```

|-------|----------|--------------|

## Цены

- Первые 1 млн символов в месяц — **бесплатно**
- Далее: 1₽ за 15 секунд аудио (~66₽/час)

Глава 11 (~10K символов) = бесплатно

## Поток мыслей (НОВОЕ)

**Юнона теперь понимает сырые мысли и сохраняет их в поток.**

### Как работает

Пишешь в бот **мысль** (не вопрос):
- "Время не движется, я движусь"
- "Маска тяжелее лица"
- "Все врут"

Юнона:
1. Сохраняет в `data/stream.jsonl`
2. Отвечает резонансом (не объяснением)

Пишешь **вопрос**:
- "Что такое ACP?"
- "Как подключиться?"

Юнона отвечает из базы знаний.
