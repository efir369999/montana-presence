# Поток мыслей Montana

> *"Твои мысли имеют вес во времени."*

---

## Как это работает

**Юнона понимает разницу между вопросом и мыслью.**

### Сырая мысль
Пишешь в бот то, что приходит в голову:
- "Время не движется, я движусь"
- "Все врут"
- "Я создаю свою игру"
- "Маска тяжелее лица"

Юнона:
1. Сохраняет мысль в поток (`data/stream.jsonl`)
2. Отвечает резонансом (не объяснением)

### Вопрос
Начинаешь с вопросительного слова или ставишь `?`:
- "Что такое ACP?"
- "Как подключиться к сети?"
- "Почему время = деньги?"

Юнона отвечает из базы знаний Montana.

---

## Формат потока

Каждая мысль сохраняется как JSON строка:

```json
{
  "user_id": 8552053404,
  "username": "junomoneta",
  "timestamp": "2026-01-18T15:17:00Z",
  "thought": "Маска тяжелее лица",
  "lang": "ru"
}
```

---

## Просмотр потока

```bash
# Последние 50 мыслей
python3 view_stream.py

# Последние 100
python3 view_stream.py 100
```

Вывод:
```
Ɉ Поток мыслей Montana (50 из 142)

[2026-01-18 15:17] @junomoneta (ru)
  Маска тяжелее лица

[2026-01-18 15:20] @junomoneta (ru)
  Время не движется, я движусь сквозь него
```

---

## Логика определения

Сырая мысль — если:
- Не заканчивается на `?`
- Не начинается с вопросительных слов (что, как, почему, когда, где, кто)
- Не начинается с команд (покажи, расскажи, объясни, помоги)
- Не длиннее 500 символов

Всё остальное — вопрос/запрос.

---

## Резонанс Юноны

Юнона не объясняет мысли. Она их отражает.

| Твоя мысль | Резонанс Юноны |
|------------|----------------|
| "Время течёт, а я стою" | "Время не течёт. Ты движешься сквозь него." |
| "Все врут" | "Ложь — это шум. Твоё присутствие — сигнал." |
| "Устал притворяться" | "Маска тяжелее лица." |

Один штрих. Отражение, не диалог.

---

## Интеграция с сетью

Поток синхронизируется между 5 узлами Montana каждые 12 секунд (дыхание сети).

```
Локал → Amsterdam → Moscow → Almaty → SPB → Novosibirsk
```

Твои мысли становятся частью когнитивной цепочки Montana.

---

## Свойства

- **Временная привязка** — каждая мысль имеет UTC timestamp
- **Неизменность** — append-only журнал (JSONL)
- **Анонимность** — username опционален
- **Многоязычность** — ru/en/zh автоопределяется

---

## Когнитивная подпись

> *"Стиль — мой. Ценности — мои. Структура мышления — моя."*

Твои мысли в потоке — это когнитивная подпись.
Не через память, а через консистентность.
Не "я помню кто я", а "я не могу быть другим".
