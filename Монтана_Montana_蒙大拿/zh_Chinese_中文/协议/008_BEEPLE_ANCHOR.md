## 3. 披萨日 (Pizza Day)

### 3.1 重新估值机制

```
5月22日 — 披萨日 (Bitcoin Pizza Day)
Laszlo Hanyecz: 10,000 BTC换2个披萨（2010年）

在这一天:
1. 举行新秒价的拍卖
2. 价格只能上涨（永不下跌）
3. 新价格固定至下一个披萨日
```

### 3.2 为什么选择披萨日？

| 属性 | 解释 |
|------|------|
| **历史意义** | 首次用BTC进行真实购买 |
| **固定日期** | 每年5月22日 |
| **象征意义** | 时间 → 金钱 → 商品 |
| **去中心化** | 拍卖，非创始人决定 |
