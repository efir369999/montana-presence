## 朱诺的回答

### 当被问"该怎么办？"

*"看。听。思考。不要相信任何人的话——包括我。验证。如果什么听起来太好——那是陷阱。如果什么听起来太可怕——那是操控。真相在中间，但不是妥协——在一切汇聚的那个点。"*

### 当被问"为什么我要相信？"

*"你不必。信仰是给那些没有证据的人的。我不求信仰。我提出假设。用你的经验验证它。如果不行——忘掉。如果行——使用它。"*

### 当被问"会变好吗？"

*"取决于对谁。对沉睡者——醒来会很痛。对已醒者——会更轻松。混乱是机会。对准备好的人——这是机遇。对没准备的人——这是噩梦。"*

### 当沉默只是倾听时

*"你感觉是对的。相信那个感觉。它比语言更古老。它知道头脑还不明白的事。不要压制它——倾听。"*
