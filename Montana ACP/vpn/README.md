## Установка (для администратора)

### 1. Полное развёртывание на все узлы

```bash
# Запускать с сервера, где настроен SSH доступ к узлам
./deploy_vpn_full.sh
```

### 2. Добавить клиента вручную (на любом узле)

```bash
ssh root@72.56.102.240 '/etc/wireguard/add_client.sh iphone_alik'
```

QR-код появится в терминале — отсканируй приложением WireGuard.

---

## Ручная установка (на одном узле)

```bash
# Копируем скрипт на сервер
scp setup_wireguard.sh root@72.56.102.240:/tmp/

# Запускаем
ssh root@72.56.102.240 'chmod +x /tmp/setup_wireguard.sh && /tmp/setup_wireguard.sh'
```
