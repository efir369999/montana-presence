## 2. ФОРМАТ ВРЕМЕНИ (UTC)

```
╔═══════════════════════════════════════════════════════════════════════════════╗
║   ЕДИНЫЙ ФОРМАТ ВРЕМЕНИ ДЛЯ ВСЕХ ДОКУМЕНТОВ                                  ║
║                                                                               ║
║   В тексте:           DD.MM.YYYY HH:MM UTC                                   ║
║   Пример:             09.01.2026 15:30 UTC                                   ║
║                                                                               ║
║   В имени файла:      DD.MM.YYYY_HH-MM_UTC_<описание>.md                     ║
║   Пример:             09.01.2026_15-30_UTC_session.md                        ║
║                                                                               ║
║   НАРУШЕНИЕ ФОРМАТА = БАН                                                    ║
╚═══════════════════════════════════════════════════════════════════════════════╝
```

### Правила UTC

1. **Всегда UTC** — никогда локальное время
2. **Время не в будущем** — относительно текущего момента
3. **Допустимая погрешность:** ±5 минут для thoughts, ±1 минута для коммитов
4. **Каждый проверяет других** — обязанность всех участников

## 5. ГОЛОСОВАНИЕ

### Правило единогласия

| Действие | Требование |
|----------|------------|
| Избрание Председателя | 100% ЗА |
| Изменение протокола | 100% ЗА |
| Исключение участника | 100% ЗА |
| Хардфорк | 100% ЗА |

### Правила

1. **Только ЗА или ПРОТИВ** — никаких "воздержался"
2. **"Не ответил"** = ПРОТИВ = ломает единогласие
3. **Срок голосования:** 24 часа
4. **Голос ПРОТИВ** требует объяснения
