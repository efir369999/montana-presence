## Ключевые решения Наблюдателя

1. **"Мои ключи — это мои мысли"**
   - Идентичность = стиль мышления, не криптография
   - Когнитивный маркер + thoughts trail = подпись

2. **"Кто UTC время неправильно ставит — тот нахуй с пляжа"**
   - Закон без исключений
   - Искажение времени = искажение реальности

3. **Когнитивный консенсус**
   - Консенсус через мысли
   - Атакующий должен украсть стиль мышления, не ключ

4. **Когнитивный промпт подписи**
   - Обязательно для входа в Совет
   - Каждый пишет свой
