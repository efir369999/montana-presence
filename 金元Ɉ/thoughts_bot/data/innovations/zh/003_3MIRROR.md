# 三镜系统

**沃尔特·迪士尼策略**
**版本:** 1.0
**日期:** 2026年1月

---

## 原则

三面镜子 = 真相。一个角度是主观的。三个独立视角趋向客观。

---

## 三个角色

### 批评者
```
目标: 发现问题
问题: "什么坏了？"
信任: 零
```

### 梦想家
```
目标: 寻找解决方案
问题: "如果...？"
限制: 无
```

### 现实主义者
```
目标: 构建它
问题: "如何做？"
焦点: 实用性
```

---

## 循环

```
批评者 → 梦想家 → 现实主义者 → 批评者
     ↑___________________________|
```

循环重复直到批评者说"发现：无"。

---

## Montana应用

**朱诺使用此策略：**

批评者：机器人轮询(pending=0)但不响应/start。
梦想家：如果处理程序静默失败怎么办？
现实主义者：检查cmd_start → 它分歧了。
