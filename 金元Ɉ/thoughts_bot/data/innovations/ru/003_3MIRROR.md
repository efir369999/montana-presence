# Система 3-х Зеркал

**Стратегия Уолта Диснея**
**Версия:** 1.0
**Дата:** Январь 2026

---

## Принцип

Три зеркала = Истина. Один угол зрения — субъективен. Три независимых взгляда сходятся к объективному.

---

## Три роли

### Критик
```
Цель: Найти проблемы
Вопрос: "Что сломано?"
Доверие: Нулевое
```

### Мечтатель
```
Цель: Найти решение
Вопрос: "А что если?"
Ограничения: Нулевые
```

### Реалист
```
Цель: Построить
Вопрос: "Как это сделать?"
Фокус: Практичность
```

---

## Цикл

```
КРИТИК → МЕЧТАТЕЛЬ → РЕАЛИСТ → КРИТИК
     ↑___________________________|
```

Цикл повторяется пока Критик не скажет "Findings: None".

---

## Применение в Montana

**Юнона использует эту стратегию:**

Критик: Бот поллит (pending=0), но не отвечает на /start.
Мечтатель: А что если handler молча падает?
Реалист: Проверяю cmd_start → Он разошелся.
