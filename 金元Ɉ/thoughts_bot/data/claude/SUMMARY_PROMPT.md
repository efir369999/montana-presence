# Промпт для создания резюме Claude диалогов

## Формат резюме

Для каждого диалога создавать резюме по следующей структуре:

```markdown
# [Номер]. [Название]

**Дата:** [дата диалога]
**Категория:** [см. ниже]
**Ключевые темы:** [3-5 тем через запятую]

---

## Краткое содержание
[2-3 абзаца о сути диалога]

## Ключевые инсайты
- [Инсайт 1]
- [Инсайт 2]
- [Инсайт 3]

## Связь с Montana Protocol
[Как этот диалог связан с развитием протокола]

## Цитаты
> "[Важная цитата 1]"
> "[Важная цитата 2]"

---
*Автор: Алик Хачатрян*
*Резюме: Claude*
```

---

## Категории

| Код | Категория | Описание |
|-----|-----------|----------|
| `PHIL` | Философия | Онтология, эпистемология, метафизика |
| `TECH` | Технология | Код, архитектура, протоколы |
| `ECON` | Экономика | Токеномика, деньги, ценность |
| `PSYCH` | Психология | Сознание, эмоции, мотивация |
| `SPIRIT` | Духовность | Религия, мистика, трансценденция |
| `SOCIAL` | Социум | Отношения, общество, политика |
| `ART` | Искусство | Творчество, эстетика, красота |
| `LIFE` | Жизнь | Личный опыт, история, путешествия |

---

## Инструкция для Claude

При создании резюме:

1. **Читай весь диалог** — не пропускай части
2. **Выделяй суть** — не копируй дословно, передавай смысл
3. **Находи связи** — как этот диалог связан с Montana
4. **Сохраняй голос** — резюме должно звучать как автор
5. **Категоризируй точно** — одна основная категория

## Пример резюме

```markdown
# 166. Симуляция

**Дата:** Октябрь 2025
**Категория:** PHIL
**Ключевые темы:** симуляция, реальность, сознание, время, доказательство

---

## Краткое содержание

Диалог начинается с вопроса о природе реальности и возможности того,
что мы живём в симуляции. Обсуждается парадокс Бострома и его импликации
для понимания времени и сознания.

Ключевой поворот происходит когда обсуждается, что доказательство
симуляции невозможно изнутри системы — это приводит к асимптотической
истине: мы можем приближаться к знанию, но никогда не достигнем
абсолютной уверенности.

## Ключевые инсайты

- Вопрос "реально ли это" бессмысленен без определения реальности
- Время — единственное неподдельное доказательство существования
- Асимптотическая истина применима не только к криптографии

## Связь с Montana Protocol

Этот диалог заложил философский фундамент для формулы
`lim(evidence → ∞) 1 Ɉ → 1 second` — идея о том, что абсолютная
истина недостижима, но практическая достаточность возможна.

## Цитаты

> "Время — единственный ресурс, который нельзя подделать.
> Можно подделать деньги, подписи, даже воспоминания.
> Но нельзя подделать прошедшие секунды."

> "Асимптота — это не ограничение. Это честность."

---
*Автор: Алик Хачатрян*
*Резюме: Claude*
```

```
Создай резюме по шаблону выше
Сохрани как [X]_[название]_SUMMARY.md
```
