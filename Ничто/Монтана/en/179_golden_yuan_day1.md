## Memory mechanisms

### 1. Hippocampus — novelty detector

Hippocampus compares incoming information with already stored. If pattern is new — encodes as separate memory. If similar to existing — either ignores or integrates into existing trace.

This is called pattern separation and pattern completion.

### 2. Predictive coding

Brain constantly predicts the next moment. Not all information is encoded, only prediction error — what differs from expected.

Routine day = few prediction errors = little is recorded = day "compresses".

### 3. Sleep consolidation

During sleep similar memories literally merge. Hippocampus "plays back" daily events, neocortex finds common patterns and saves generalization instead of each separate episode.

I call this disk defragmentation, I've known since my first computer, Dad bought it on credit.
