Политика конфиденциальности

Последнее обновление: 28 января 2026

Zero Collection по умолчанию
Montana Protocol не собирает персональные данные. Когнитивный ключ не хранится — он используется для генерации ключей и сразу стирается из памяти.

1. Какие данные мы НЕ собираем

  • Когнитивные ключи — не хранятся вообще (стираются из памяти после генерации ключей)
  • Приватные ключи ML-DSA-65 — генерируются и хранятся только локально в Keychain
  • Историю транзакций с привязкой к личности
  • IP-адреса для идентификации
  • Геолокацию
  • Контакты, фото, сообщения

2. Какие данные обрабатываются

Для работы сети Montana обрабатываются:

  • Публичный ключ — для верификации подписей (не содержит персональных данных)
  • Montana адрес (mt...) — криптографический хеш публичного ключа
  • Подписи транзакций — ML-DSA-65 подписи для подтверждения операций

Эти данные являются публичными и не позволяют идентифицировать личность.

3. Локальное хранение

На вашем устройстве хранятся:

  • Приватный ключ ML-DSA-65 (в Keychain с биометрической защитой)
  • Публичный ключ и адрес
  • Локальный кеш баланса

Когнитивный ключ НЕ хранится — он используется один раз для генерации ключей и сразу стирается из памяти. Для восстановления на другом устройстве вы должны помнить его сами.

Все данные зашифрованы средствами iOS и защищены биометрией.

4. Постквантовая криптография

Montana использует ML-DSA-65 (FIPS 204) — постквантовый алгоритм подписи, устойчивый к атакам квантовых компьютеров.

  • Приватный ключ: 4032 байта
  • Публичный ключ: 1952 байта
  • Подпись: 3309 байт

5. Ваши права

  • Полный контроль — только вы владеете когнитивным ключом
  • Удаление — удалите приложение, и все локальные данные будут стёрты
  • Переносимость — когнитивный ключ позволяет восстановить кошелёк на любом устройстве
  • Анонимность — Montana адрес не связан с личностью

6. Сервер не нужен

Для восстановления кошелька сервер НЕ нужен. Когнитивный ключ детерминистически генерирует одни и те же ключи ML-DSA-65 на любом устройстве.

"Тот же ключ = тот же кошелёк, всегда."
