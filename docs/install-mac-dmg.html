💻 Установка DMG на Mac

Классический способ установки

⬇ Скачать DMG

Montana_2.0.0.dmg · 72 KB

1 Скачай DMG файл
Нажми кнопку выше. Файл сохранится в папку Загрузки.
2 Открой DMG
Дважды кликни на скачанный файл Montana_2.0.0.dmg. Откроется окно с приложением Montana.
3 Перетащи в Applications
Перетащи Montana.app в папку Applications (или нажми на стрелку в окне DMG).
4 Открой Терминал
Finder → Программы → Утилиты → Terminal
(или Spotlight: ⌘+Space, напиши "Terminal", Enter)
5 Выполни команду
Скопируй и вставь эту команду:
sudo xattr -cr /Applications/Montana.app && open /Applications/Montana.app
Введи пароль от Mac (символы не отображаются) и нажми Enter.
6 Готово!
Montana откроется автоматически. В следующий раз можно открывать как обычное приложение.

⚠ Почему нужен Терминал?

Apple блокирует все приложения без платной подписи Developer ID ($99/год). Montana Protocol — open source и бесплатный. Команда xattr -cr снимает карантин. Исходный код открыт.
