Терминал (Mac / Linux)

CLI + сервис присутствия

▼

Одна команда устанавливает Montana CLI и сервис присутствия (launchd/systemd). На Mac также устанавливается GUI приложение. На Linux — только терминал.

🔒 Self-Custody Криптография

Приватный ключ ML-DSA-65 генерируется локально из твоего когнитивного ключа и НИКОГДА не покидает устройство. PBKDF2 с 600,000 итерациями защищает от brute-force. Подпись устойчива к квантовым компьютерам (FIPS 204). Исходный код открыт.
