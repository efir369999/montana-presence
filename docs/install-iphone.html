1 Установи AltStore

Скачай AltStore на компьютер (Mac/Windows). Запусти AltServer и установи AltStore на iPhone через USB.

2 Добавь источник Montana

Открой эту страницу на iPhone и нажми кнопку «Добавить в AltStore». Или вручную:

https://efir.org/altstore-source.json

3 Установи Montana

AltStore → Sources → Montana Protocol → Install. Приложение появится на домашнем экране.

💡 AltStore автоматически обновляет приложение каждые 7 дней. AltServer должен быть запущен на компьютере в одной Wi-Fi сети.

⚙ Альтернатива: Sideloadly

Скачай IPA и установи через Sideloadly — подключи iPhone, выбери IPA, нажми Start.
