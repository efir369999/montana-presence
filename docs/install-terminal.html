CLI + Сервис присутствия для Mac и Linux

Linux
1 Открой Терминал
Cmd + Пробел → введи "Terminal" → Enter
2 Выполни команду установки
Скопируй команду выше и вставь в Терминал. Нажми Enter. Введи пароль администратора когда попросит.
3 Дождись завершения
Скрипт установит:
• Montana CLI в /usr/local/bin/montana
• GUI приложение в /Applications/Montana.app
• Сервис присутствия (launchd)
4 Создай кошелёк
В Терминале выполни:
montana init
Придумай когнитивный ключ — уникальную фразу минимум из 24 слов. Это твоя история. Только ты её знаешь.
1 Открой терминал
Используй свой любимый терминал (gnome-terminal, konsole, xterm и т.д.)
2 Выполни команду установки
Скопируй команду выше и выполни. Потребуется sudo.
3 Дождись завершения
Скрипт установит:
• Montana CLI в /usr/local/bin/montana
• Systemd сервис присутствия
• БЕЗ GUI — всё через командную строку
4 Создай кошелёк
montana init
5 Проверь сервис
systemctl status montana-presence

Команды Montana CLI

montana init Создать кошелёк
montana restore Восстановить из ключа
montana balance Проверить баланс
montana send <addr> <amt> Перевод
montana status Статус сервиса
montana --help Справка

🔒 Синхронизация между устройствами

Один когнитивный ключ = один адрес на всех устройствах. Создай кошелёк на Mac, затем используй montana restore на Linux с тем же ключом — получишь тот же адрес и баланс.
