# Montana Style Guide

Reference PDF: `Montana_style_template.pdf`

## Style Rules

### Typography
- **Font:** Times New Roman
- **Title:** Centered, large
- **Author/Email/Date:** Centered, separate lines
- **Body:** Justified text

### Layout
- Academic paper format (like Bitcoin whitepaper)
- Clean margins
- No duplicate titles
- Abstract with margins, centered

### Symbol
- Currency symbol: **Éˆ** (U+0248, Latin Capital Letter J with Stroke)
- Must render correctly in PDF

### Structure
```
Title (centered)
Author
Email
Date

Abstract (centered, with margins)

1. Section
2. Section
...

References
Appendix
```

### Generation
Use weasyprint with custom CSS for Times New Roman and proper Unicode support.
